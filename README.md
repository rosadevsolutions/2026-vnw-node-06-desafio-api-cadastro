### Instalação e Inicialização do Projeto

Através do terminal de sua escolha:

1. **Download/Clonar o projeto para o seu dispositivo local:**<br>
   [Repositório Remoto no Github](https://github.com/rosadevsolutions/2026-vnw-node-06-desafio-api-cadastro)<br>

2. **Acessar o projeto:**<br>
   Acesse a pasta raiz na qual o projeto foi clonado

3. **Instalar dependências de desenvolvimento:**<br>
   Digitar no terminal o comando `npm install`

4. **Inicializar/Rodar o servidor de desenvolvimento do projeto:**<br>
   Digitar no terminal o comando `npm run dev`

5. **Visualizar o projeto**:<br>
   Acessar o link gerado após inicializar o projeto. Copie o link e abra em seu navegador de preferência

---

### Ferramentas NPM utilizadas no projeto

1. **Express**
   Framework com estrutura web performática, flexível e minimalista dedicada a aplicativos Node JS.<br>
   Comando de Instalação: `$ npm install express`

2. **Nodemon**
   Pacote de dependêcia de desenvolvimento que monitora alterações nos arquivos do projeto e reinicia automaticamente o servidor, eliminando a necessidade de reinicializações manuais.<br>
   Comando de Instalação: `$ npm install nodemon --save-dev`

3. **Dotenv**
   Pacote de dependêcia que carrega variáveis de ambiente a partir de um arquivo .env para dentro do processo da aplicação.<br>
   Comando de Instalação: `$ npm install dotenv`
